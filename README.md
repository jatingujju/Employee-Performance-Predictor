# 🚀 Employee Performance Predictor using Data Analytics

## 📌 Overview

This project predicts employee performance (Low, Medium, High) using Machine Learning and HR analytics.

It helps HR teams make data-driven decisions for promotions, training, and employee retention.

---

## 🧠 Problem Statement

Predict employee performance based on historical data such as experience, training, satisfaction, and past ratings.

---

## ⚙️ Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib
* Streamlit

---

## 🔄 Project Pipeline

1. Data Generation (Synthetic HR Dataset)
2. Data Preprocessing (Cleaning + Encoding)
3. Stratified Train-Test Split
4. Model Training (Random Forest)
5. Feature Importance Analysis
6. Streamlit Web App Deployment

---

## 📊 Results

* Achieved ~90% accuracy
* Balanced multi-class classification
* Reliable predictions using stratified sampling

---

## 📈 Key Insights

* Past performance is the strongest predictor
* Training hours significantly improve performance
* Work environment impacts productivity
* Experience and project exposure matter

---

## 💡 Business Impact

* Identify high-performing employees
* Support low performers with training
* Enable data-driven HR decisions
* Improve employee retention strategies

---
